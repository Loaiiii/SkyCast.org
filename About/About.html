About Our Weather System

Our weather system collects data from several sources to provide the most accurate and comprehensive weather information possible.

Data Sources

Sensors

This station is built using a Raspberry Pi 3 Model B and two sensors: DHT22 (temperature and humidity) and BMP180 (barometric pressure). Sensor readings are sent to a Firebase Realtime Database.

Weather API

We use Weather API to provide additional weather data for our system.

OpenWeather

We also incorporate data from OpenWeather to ensure comprehensive weather information.
